Δήλωση εκτέλεσης απαιτούμενων ενεργειών

Παρακαλώ δηλώστε τις ενέργειες που εκτελέσατε.

1) Υλοτομία και απομάκρυνση των ξερών και σπασμένων δέντρων και κλαδιών, καθώς και των κλαδιών που βρίσκονται σε άμεση επαφή με κτίσμα.

0 / 2000

2) Απομάκρυνση της καύσιμης φυτικής ύλης που βρίσκεται στην επιφάνεια του εδάφους όπως ενδεικτικά το φυλλόστρωμα, τα ξερά κλαδιά, τα κατακείμενα ξερά κλαδιά και τα ξερά χόρτα.

0 / 2000

3)Αποκλάδωση της βάσης κόμης των δέντρων και αύξηση του ύψους έναρξης της από την επιφάνεια του εδάφους, ανάλογα με την ηλικία και το είδος του δέντρου.

0 / 2000

4)Αραίωση της θαμνώδους βλάστησης ως προς την κάλυψη του εδάφους

0 / 2000

5)Απομάκρυνση τυχόν άλλων εγκαταλελειμμένων κοστών, αναφλέξιμων, εκρήξιμων ή εύφλεκτων υλικών, αντικειμένων απορριμάτων.

0 / 2000

6)Ασφάλη συλλογή και μεταφορά όλων των υπολειμμάτων Καθαρισμού.

0 / 2000
